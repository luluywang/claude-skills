# Session Logging

## Storage Locations
- Session logs: `quality_reports/session_logs/YYYY-MM-DD_description.md`
- Quality reports: `quality_reports/merges/YYYY-MM-DD_[branch-name].md`

## Templates
- `templates/session-log.md`
- `templates/quality-report.md`

## The Three Proactive Triggers

1. **Post-Plan** -- Immediately after plan approval, capture goal, approach, rationale, and context.
2. **Incremental** -- Add brief entries (1-3 lines) on design decisions, problem resolutions, user feedback, strategy shifts. Do not batch.
3. **Session Wrap-Up** -- Record summary, quality scores, unresolved questions, blockers.

## Quality Reports
Generated only at merge time, not per-commit. Use quality-report template.
