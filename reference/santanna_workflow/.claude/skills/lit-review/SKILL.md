# /lit-review [topic] -- Literature Search & Synthesis

Systematically search, synthesize, and organize academic literature on a specified topic.

## Workflow
1. Parse research topic
2. Search available resources (local paper archives, web, working paper repos)
3. Organize findings: theoretical contributions, empirical results, methodological innovations, debates
4. Identify research gaps
5. Output markdown report to `quality_reports/`

## Output Structure
- Summary
- Annotated key papers
- Thematic organization
- Identified gaps
- Next steps
- BibTeX citations

## Constraints
- Never fabricate citations; flag uncertain references
- Prioritize recent work (5-10 years) unless seminal
- Distinguish working papers from published research
- Be transparent about verification limitations

## Tools: Read, Grep, Glob, Write, WebSearch, WebFetch
